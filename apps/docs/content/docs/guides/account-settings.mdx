## Accessing Account Settings

Click on your profile in the top right corner and select **Account** to access your personal account settings.

## Profile

### Personal Information

- **Name**: Update your display name
- **Email**: Change your email address
- **Profile Picture**: Upload or change your profile picture

### Updating Your Profile

1. Go to **Account** → **Profile** tab
2. Edit your information
3. Click **Save** to apply changes

## Security

### Changing Your Password

1. Go to **Account** → **Settings** tab
2. Enter your current password
3. Enter your new password
4. Confirm your new password
5. Click **Change Password**

### Active Sessions

View and manage your active sessions:

- **Current device**: See where you're currently logged in
- **Other devices**: View all other active sessions
- **Revoke sessions**: Log out from other devices for security

### Two-Factor Authentication

If available, enable two-factor authentication for additional security:
- Set up using an authenticator app
- Use backup codes for account recovery
- Keep backup codes in a safe place

## Best Practices

### Security

- **Strong passwords**: Use a strong, unique password
- **Regular updates**: Change your password periodically
- **Review sessions**: Regularly check active sessions
- **Revoke suspicious sessions**: Log out from unknown devices

### Profile

- **Keep updated**: Keep your profile information current
- **Professional photo**: Use a professional profile picture
- **Consistent information**: Use the same information across platforms

## Tips & Tricks

1. **Password manager**: Use a password manager for secure passwords
2. **Session management**: Regularly review and revoke old sessions
3. **Profile consistency**: Keep your profile consistent across services
4. **Security alerts**: Enable notifications for security events
